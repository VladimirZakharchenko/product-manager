# Product Manager

Административная панель для управления товарами с авторизацией и интерактивной таблицей.

## Стек технологий

- **React 18+** - библиотека для пользовательских интерфейсов
- **TypeScript** - строгая типизация
- **Zustand** - управление состоянием
- **Ant Design** - UI библиотека компонентов
- **Axios** - HTTP клиент
- **React Hot Toast** - уведомления

## Функциональность

### Авторизация
- Форма входа с валидацией полей
- Тестовые данные: `emilys` / `emilyspass`
- Сохранение сессии:
  - С чекбоксом "Запомнить данные" - в localStorage (постоянно)
  - Без чекбокса - в sessionStorage (до закрытия вкладки)
- Обработка ошибок авторизации

### Управление товарами
- Загрузка товаров с DummyJSON API
- Таблица с сортировкой по столбцам:
  - Наименование (с категорией под названием)
  - Вендор
  - Артикул
  - Оценка (подсветка красным при рейтинге < 3)
  - Цена
- Поиск товаров с debounce (500ms)
- Пагинация (20 товаров на странице)
- Добавление новых товаров (локально)
- Выбор товаров чекбоксами
